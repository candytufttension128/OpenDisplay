OpenDisplay

Open-source macOS display manager. Control brightness, resolution, window tiling, and more. Free forever.

Download GitHub
🖥

DDC/CI Control

Hardware brightness, contrast, volume, sharpness, input switching, and power control. Native Apple Silicon support.

🎛

Resolution & Refresh

Access all resolutions including hidden HiDPI modes. Quick refresh rate switching. Display arrangement editor.

🌙

Night Shift & Dimming

Scheduled color temperature. Gamma and overlay dimming — dim to complete black. XDR/HDR brightness unlock.

🪟

Window Tiling

Edge snapping, corner tiling, grid layouts. Auto-tile all windows. Right-click menu bar for quick presets.

⚡

Automation

Full CLI, URL scheme, display profiles with auto-apply on connect, global hotkeys, settings export/import.

🪶

Lightweight

Native Swift. No Electron. Under 1MB. Runs as a menu bar app with zero dock clutter.

Install

# Homebrew
brew tap sahilmishra0012/opendisplay
brew install --cask opendisplay
# Build from source
git clone https://github.com/sahilmishra0012/OpenDisplay.git
cd OpenDisplay && swift run

Or download the DMG directly.
